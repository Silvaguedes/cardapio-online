Criei um cardápio digital funcional com integração ao WhatsApp, onde usuários podem montar pedidos com carrinho dinâmico e controle de quantidade. Implementei armazenamento local LocalStorage e um sistema de cadastro de produtos, permitindo que comerciantes gerenciem o cardápio sem editar código.Projeto focado em resolver um problema real de pequenos negócios, aplicando JavaScript na prática.
